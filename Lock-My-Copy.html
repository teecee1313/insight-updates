🦋 INSIGHT TRADING

🔒 Lock My Copy

Bake your personal code into the app so every device you open it on just asks for that one code — with no way for anyone to create a different one or take it over. For your own and family use.
1
Choose your master app file (insight-trading.html).
Pick the newest insight-trading.html from your Downloads. This tool never uploads it anywhere — everything happens on your device.
2
Type the code you want to use. Use the same code you already use.
The code itself is never stored in the file — only a one-way scrambled fingerprint (PBKDF2-SHA-256, 150,000 rounds). Nobody can read your code out of the file, but the right code will always unlock it.
3
Download your locked copy and use it on every device.
How to use it afterwards
• The download is named insight-trading-locked.html. Put that file on any computer, phone or tablet you like — it will always ask for your code and open with it.
• There is no "create a password" screen and no reset-to-setup — the code is fixed in the file. To change it later, just run this tool again on a fresh insight-trading.html with a new code.
• Keep one un-locked insight-trading.html as your working master for future updates; lock a copy whenever you want to deploy.
